## 1. What are Visual Features?

In the context of Visual SLAM, visual features are distinctive and repeatable points, lines, or regions in an image that can be reliably detected and tracked across different viewpoints. These features serve as landmarks that the SLAM system uses to estimate its own motion and to build a map of the environment.

## 2. Types of Features

Different algorithms extract different types of features:
- **Corners:** High-contrast points where two edges meet (e.g., Harris Corner Detector, Shi-Tomasi).
- **Edges:** Boundaries between regions of different intensity or color.
- **Blobs:** Regions of interest that stand out from their surroundings.

## 3. Feature Descriptors

A feature descriptor is a vector that describes the local appearance of a feature. It should be robust to changes in illumination, viewpoint, and scale, allowing the same feature to be recognized even if its appearance changes slightly.

Common descriptors include:
- **SIFT (Scale-Invariant Feature Transform):** Highly robust, but computationally expensive.
- **SURF (Speeded Up Robust Features):** A faster alternative to SIFT.
- **ORB (Oriented FAST and Rotated BRIEF):** A fast and efficient binary descriptor.
